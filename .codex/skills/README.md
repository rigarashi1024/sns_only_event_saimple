# Repo Skills

このディレクトリは、このリポジトリ向けに管理する skill の置き場所です。

## 例

- `.codex/skills/my-skill/SKILL.md`

## 基本ルール

- 1 skill につき 1 ディレクトリを作る
- skill 本体の説明は `SKILL.md` に書く
- 追加の資料が必要なら skill 配下に `references/` や `assets/` を置く
- スクリプトが必要なら skill 配下に `scripts/` を置く

## 推奨構成

```text
.codex/skills/
  my-skill/
    SKILL.md
    references/
    scripts/
    assets/
```
